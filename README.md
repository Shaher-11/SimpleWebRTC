# SimpleWebRTC

>SimpleWebRTC is a simple video chat app built with vanilla javascript and node.js


## Built With

- JavaScript
- Node.js
- HAML
- Semantic UI
- Lots of love :heart:

## Get started

- open your terminal 
- run ```git clone https://github.com/Shaher-11/SimpleWebRTC```
- run ``npm install``
- run ``npm start``
- open your browser and go to [localhost:3000](http://localhost:3000)
